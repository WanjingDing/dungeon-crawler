# AI Usage Documentation

## Tools Used
This project used the generative AI tool: ChatGPT

---

## How AI Was Used

AI was used to assist in the development of this project in the following ways:

- Helping design the overall structure of the project (models, engine, commands, CLI)
- Debugging errors in Python code
- Suggesting improvements to game logic
- Improving documentation, including README.md

---

## AI-Generated Outputs

AI generated or suggested:

- Example implementations for functions (e.g., command parsing, CLI loop)
- Sample test cases for pytest
- Code structure suggestions (modular design)
---

## Human Review and Modifications

All AI-generated content was:

- Carefully reviewed by the authors
- Modified to fit the project requirements
- Tested to ensure correctness
- Integrated with manually written code

No AI-generated code was used without verification.

---

## Summary

AI was used as a support tool to improve efficiency and understanding, but all final decisions, implementations, and validations were performed by the project authors.
